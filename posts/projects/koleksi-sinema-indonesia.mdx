Setiap petualangan dimulai dari sebuah langkah kecil, begitu juga dengan
petualangan saya di dunia pengembangan perangkat lunak web yang sangat cepat
ini, salah satu serangkaian langkah-langkah kecil saya adalah mengikuti salah
satu kursus yang dikembangkan oleh dicoding yaitu Belajar Fundamental
Front-End Web Development, seperti dengan nama kursusnya sendiri, tentu saja
materinya berisi mengenai hal-hal mendasar yang berkaitan dengan frontend
development seperti transaksi data melalui protokol HTTP/HTTPS menggunakan
fetch API, penggunaan Webpack untuk membantu dalam _development_ dan
_production_, dan pengembangan komponen antar pengguna yang bersifat
_reusable_ dan _modular_. Untuk menguji pemahaman mengenai materi yang sudah
dipelajari, maka setiap pembelajar diwajibkan untuk mengembangkan sebuah
website dengan berbagai aturan yang harus diterapkan

## Apa

Sinema Indonesia adalah website yang digunakan untuk mencari sinema-sinema
karya para sineas Indonesia, dibangun menggunakan TMBD API sebagai sumber data
dinamis, web component agar komponen antarmuka pengguna lebih _reusable_ dan
_modular_, Tailwind CSS untuk _styling_ setiap komponen, dan Webpack yang
digunakan untuk membantu _development_ dan juga untuk mem-_bundle_ aplikasi
agar optimal dan bisa di-_deploy_ ke sebuah _server_, dalam hal ini yang saya
gunakan adalah Vercel.

## Kenapa

Alasan saya akhirnya memilih mengembangkan proyek ini adalah karena sejauh
yang saya tahu, di Indonesia sendiri belum ada laman, maupun data daring
(dalam jaringan) yang menyediakan informasi mengenai sinema-sinema karya
sineas Indonesia.

## Proses

Proses pengembangan aplikasi ini kira-kira memakan waktu kurang lebih 2 minggu
dan menghabiskan beberapa gelas es teh yang tak terhitung jumlahnya.

### First Step

Langkah pertama yang saya lakukan setelah mengetahui requirement yang
dibutuhkan untuk menyelesaikan kelas tersebut adalah menentukan apa saja yang
akan digunakan, pertimbangannya adalah library ataupun tools yang akan
digunakan bisa dipelajari dalam waktu singkat, berikut beberapa tools/library
yang akhirnya saya gunakan selain Webpack beserta loader dan pluginnya:

1. [The Movie Database (TMDB) API](https://www.themoviedb.org/)

   TMDB adalah database film dan TV yang dibuat oleh komunitas. Setiap bagian
   data telah ditambahkan oleh komunitas sejak tahun 2008. API ini dipilih
   karena sesuai dengan [kenapa](#kenapa) saya akhirnya memilih mengembangkan
   proyek ini. TMBD API sendiri memiliki
   [dokumentasi](https://developers.themoviedb.org/3/getting-started/introduction)
   dan examples yang baik yang membuatnya mudah dipelajari oleh pemula seperti
   saya ini.

2. [Tailwind CSS](https://tailwindcss.com/)

   Dikutip dari laman resminya, Tailwind CSS adalah

   > A utility-first CSS framework packed with classes like **flex**,
   > **pt-4**, **text-center** and **rotate-90** that can be composed to build
   > any design, directly in your markup.

   Sebenarnya, penggunaan Tailwind CSS bukanlah sebuah kewajiban untuk
   menyelesaikan submisi ini, namun karena saat itu saya tidak ingin
   menggunakan Bootstrap dan saya cukup tertarik untuk mencobanya dan
   menggunakanya, maka saya putuskan untuk menggunakan Tailwind CSS walaupun
   memiliki learning curve yang cukup _steep_, tapi itu bukan masalah karena
   Tailwind CSS menyediakan [dokumentasi](https://tailwindcss.com/docs) yang
   sangat baik dan
   [course](https://www.youtube.com/playlist?list=PL7CcGwsqRpSM3w9BT_21tUU8JN2SnyckR)
   yang sangat membantu bagi para pendatang baru yang ingin mempelajari
   Tailwind CSS melalui video

### Development

Selanjutnya dimulailah tahapan _development_ diawali dengan konfigurasi
Webpack untuk _development_ dan _production_, lalu konfigurasi Tailwind CSS
agar bisa digunakan, membuat komponen antarmuka-pengguna menggunakan HTML
templates, Custom elements dan Shadow DOM, mengambil data dari TMDB API
menggunakan Fetch API, dan terakhir mem-_bundle_ aplikasi agar optimal dan
bisa dideploy ke sebuah _server_, dan bisa digunakan oleh para pengguna.
